# Отзывы для сайта

**Источник правды:** папка **«Все отзывы»** в публичном портфолио на [Яндекс.Диске](https://disk.yandex.ru/d/ZDM9-AR8LwjYWw) (файлы `.docx`).

При каждом `npm run build` и `npm run dev` скрипт `scripts/sync-reviews.mjs` скачивает эти документы и собирает **`data/reviews.json`**.

## Локальный запасной вариант

Если Яндекс недоступен при сборке, можно положить сюда `.md` / `.txt` (см. старый формат в истории git). Файлы `README*` не учитываются.

## Переменные окружения (опционально)

- `YANDEX_PORTFOLIO_PUBLIC` — другая публичная ссылка на корень портфолио.
- `SKIP_YANDEX_REVIEWS=1` — не ходить в API, только `content/reviews`.
