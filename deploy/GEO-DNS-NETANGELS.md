# Два сервера (Москва + Европа) и домен на NetAngels

Цель: пользователям из РФ отдавать **IP Москвы**, остальному миру — **IP европейского VPS**. Один домен (`truewebwork.ru`), один сайт в коде. **Шаг 3 (DNS)** вы настраиваете в панели NetAngels; здесь — что выбрать и как связать с деплоем.

## Этап 1–2 (уже в репозитории)

- Верификация: переменные `NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION` и `NEXT_PUBLIC_YANDEX_VERIFICATION` в `.env` на **каждом** сервере (одинаковые токены).
- `robots.txt` открывает индексацию для всех и для Yandex, админку закрывает.
- Деплой:
  - `npm run deploy:vps` или `npm run deploy:vps:eu` — европейский VPS (`TRUWEB_VPS_HOST`).
  - `npm run deploy:vps:ru` — московский VPS (`TRUWEB_VPS_HOST_RU` в `.deploy.env`).

## Этап 3: NetAngels (ваша часть)

1. Купить VPS в РФ, зафиксировать **публичный IPv4**.
2. Один раз выполнить `npm run deploy:vps:ru` с заполненным `.deploy.env` (см. `.deploy.env.example`).
3. В NetAngels для зоны `truewebwork.ru` включить **геозависимые записи** (если услуга называется «GeoDNS», «записи по регионам» или аналог — смотрите справку NetAngels).

   Типичная схема:

   - **Россия / СНГ (или «РФ»)** → A-запись на IP **Москвы**.
   - **По умолчанию (остальной мир)** → A-запись на IP **Австрии**.

   Важно: не делайте два разных «сайта» с разным контентом без продуманного `canonical` — у вас один код, два зеркала с одной сборкой.

4. TTL можно поставить 300–600 секунд на время проверки, потом поднять.

## SQLite и два сервера

Сейчас БД — **файл на каждом сервере**. Два деплоя = **две независимые базы** (заказы/админка на EU и RU разъедутся). Варианты на будущее:

- одна **общая** БД (PostgreSQL и т.д.) в одном месте;
- или основной приём заказов только на одном узле (сложнее в nginx).

Пока витрина и FAQ работают с минимальной записью — осознайте риск для корзины/ЛК.

## Про «Яндекс vs Google»

DNS смотрит **географию резолвера / пользователя**, а не «из какого поиска пришли». Для клиентов из России и Яндекс, и Google обычно откроют сайт с **российского IP** — это нормально и ожидаемо.
